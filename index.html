slogan text

We provide development and design services for modern web applications.
We focus on embracing good ideas and delivering elegant solutions.

  • Pozit

    Pozit is a simple tool that enables you to ask your friends a question (via email, facebook, twitter, etc) with a link to a shortened URL. This "pozit" serves as an invite-only discussion space where your friends can give their recommendations and make comments.

    See Project

  • safetySTREAM

    safetySTREAM is an innovative, simple solution for safety risk management. This app was built with CS2 Technologies for use in construction sites around the world. Users can report on incidents that occur real time in the field via their hand held devices.

    The web side of the app communicates seamlessly with the hand-held device to allow users in the field to coordinate with administrative users, providing the best proactive risk management possible.

    See Project

  • Shortbord

    Shortbord is probably the best way to advertise on the web. Social media users ("surfers") sign endorsement deals with companies ("sponsors") to display the company's logo next to the surfer's avatar on social media sites ("partners").

    thoughtbot worked with the three founders of Shortbord to design and build a Rails app from scratch. As part of our Kickstart program, the founders work out of our Boston office and we help them find and train technical hires.

    See Project

  • Gemcutter

    Gemcutter is the new gem hosting solution for the ruby community. It provides a modern API for dealing with gem management, transparent and accessible project pages, and is 100% open source code, allowing the ruby community to improve the site. thoughtbot contributed development time and created the graphical design for the site.

    See Project
