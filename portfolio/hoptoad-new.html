Billions of errors.
One Toad.

Hoptoad — the original “app error app” — is a software as service (SaaS) style web application operated by thoughtbot. The service collects errors generated by other web applications, and aggregates the results for developer review. It has helped thousands of developers sort through millions of problems, and improve the quality of their own web applications.
